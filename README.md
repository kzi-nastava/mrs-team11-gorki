mrs-team11-gorki is a full-stack application consisting of:

- Angular web frontend (HTML, CSS, TypeScript)
- Mobile client(s)
- Spring Boot backend (REST API)

The system is built as a modular client-server architecture where both web and mobile applications communicate with the backend via REST APIs.

## Tech Stack

### Web Frontend
- Angular
- TypeScript
- HTML5
- CSS3
- Angular Router
- RxJS

### Mobile
- Mobile client application (Android)
- REST API communication with backend

### Backend
- Java
- Spring Boot
- Spring Web (REST Controllers)
- Spring Data JPA (Hibernate)
- Spring Security
- Maven
- Database (PostgreSQL)


## Prerequisites

Make sure you have installed:

### For Frontend
- Node.js (LTS recommended)
- npm
- Angular CLI

### For Backend
- Java 17+
- Maven
- PostgreSQL
